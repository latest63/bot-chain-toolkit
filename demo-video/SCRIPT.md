# BOT Chain Toolkit — Demo Video Script

**Theme:** Communities coming to BOT Chain need tools to grow. The Toolkit is ready.
**Tone:** Bold, cinematic, aspirational. Dark theme. Premium product demo.
**Duration:** ~60-75 seconds
**Music:** Deep cinematic build-up → uplifting swells → confident resolve

---

## Scene 1 — The Invitation (0:00–0:15)

| Element | Description |
|---|---|
| **Visual** | Slow zoom into BOT Chain logo on dark grid. Particles floating. Text fades in: "BOT CHAIN" → "COMMUNITIES ARE ARRIVING" |
| **Narration** | "BOT Chain is live. New communities are arriving every day. But infrastructure alone isn't enough — communities need **tools** to distribute, engage, and grow." |
| **Transition** | Logo drifts up, screen fades to dark, toolkit branding emerges |

---

## Scene 2 — Batch Splitter (0:15–0:32)

| Element | Description |
|---|---|
| **Visual** | Toolkit interface slides in from the left. Dark UI with teal accent (#00D4AA). Shows the Batch Splitter tool: paste multiple addresses, enter amount, see gas savings counter. Transaction animation fires — green success pulse. |
| **Narration** | "Meet the Batch Splitter. Send BOT to your entire community in a **single transaction**. Token airdrops, reward distributions, payroll — without the gas nightmare. One click. Everyone paid." |
| **On-screen** | "BATCH SPLITTER" label badge. Address list populates with fake addresses. "SEND 0.5 BOT TO 12 ADDRESSES" → TX SUCCESS. Counter: "Gas saved: 91%" |
| **Transition** | Interface slides right, the Raffle banner slides in |

---

## Scene 3 — Raffle System (0:32–0:50)

| Element | Description |
|---|---|
| **Visual** | Raffle interface with a vibrant banner (#00D4AA glow). Shows: create raffle → set ticket price → users buy → draw winner animation (blockhash random) → winner announcement with confetti burst. |
| **Narration** | "The Raffle System. Launch community giveaways in seconds. Set your rules, sell tickets, and let the chain pick the winner — provably fair, transparent, and permissionless. Hype your community, reward your loyal members." |
| **On-screen** | Raffle creation form fills itself. Ticket count ticks up. "DRAW WINNER" button pulses. Winner address reveals with trophy animation. Prize pool counter. |
| **Transition** | Zoom out to show both tools side by side on a dashboard |

---

## Scene 4 — Call to Action (0:50–1:05)

| Element | Description |
|---|---|
| **Visual** | Full toolkit dashboard emerges — both tools visible, "MORE TOOLS COMING" badge with a glowing pulse, stats counter showing "1,247 BOT distributed • 89 raffles created". BOT Chain logo + Toolkit logo lockup at bottom. |
| **Narration** | "BOT Chain Toolkit. Built for the communities building on BOT Chain. Two tools today. Many more tomorrow. If you're bringing a community to BOT Chain, this is your launchpad." |
| **End card** | "BOT CHAIN TOOLKIT" + "Launch. Distribute. Grow." + "botchain-toolkit.vercel.app" (or wherever it's deployed) |

---

## Production Notes

- Use dark background throughout (#0a0a0f → #111118 gradient)
- Accent: #00D4AA (teal) for CTAs, highlights, borders
- Secondary: #7C7C8A for muted text
- UI font: Inter or SF Pro Display for all on-screen elements
- Transitions: smooth cross-fades, slide motions, subtle glows
- No stock footage — all UI mockups or screen recordings of the actual app
